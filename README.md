## Instructions

### Installation

- Run oil installer

    $ curl get.fuelphp.com/oil | sh
    
- Clone this repository

    $ git clone https://github.com/moakdesigns/SynergyNetworksCenter.git
    
- Run composer.phar

    $ php oil composer.phar update
    $ php oil composer.phar self-update

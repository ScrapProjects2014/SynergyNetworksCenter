## Instructions

### Installation

- Clone this repository

    $ git clone https://github.com/moakdesigns/SynergyNetworksCenter.git

- Run oil installer

    $ curl get.fuelphp.com/oil | sh    

- Run composer.phar

    $ oil composer.phar update  
    $ oil composer.phar self-update
